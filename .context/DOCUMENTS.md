# Command Reference

## Setup
- `npm install` — install extension dependencies.

## Build
- `npm run compile` — build the extension once.
- `npm run watch` — incremental rebuild during development.

## Package/Test
- `npm run lint` — placeholder (no lint configured).
- `npm test` — placeholder (no tests yet).

## Run / Debug
- In VS Code, choose the `Run Extension` launch config and press F5 (or Run → Start Debugging) to open an Extension Development Host. If prompted for a debug type, pick “VS Code Extension” instead of Markdown.
